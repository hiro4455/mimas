MIMAS
============================================================

MIMASはTypeScript上で実装された簡易スクリプトです。

ゲーム制作で、非プログラマが会話劇を記述する用途で作成されたもので、
複雑な構文等はサポートしていません。


組み込み方法
------------------------------

src以下のtsファイルをコピーして対象プロジェクトに加えて下さい。



テスト方法
------------------------------

### 準備

```
$ npm install
```

### ビルド

```
$ npm run build
```

### 実行

```
$ npm run start
```
